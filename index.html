Anthronim

NVIDIA NIM uç noktasını Anthropic Messages API olarak sunan proxy

Bu proxy'ye erişim anahtarı gereklidir.

Erişim anahtarı almak için x.com/koroglan adresinden mesaj gönderin.

Claude Code Yapılandırması

~/.claude/settings.json dosyasına ekleyin:

{
  "env": {
    "ANTHROPIC_BASE_URL": "",
    "ANTHROPIC_API_KEY": "<erişim anahtarınız>",
    "ANTHROPIC_DEFAULT_OPUS_MODEL": "minimaxai/minimax-m2.5",
    "ANTHROPIC_DEFAULT_SONNET_MODEL": "minimaxai/minimax-m2.5",
    "ANTHROPIC_DEFAULT_HAIKU_MODEL": "z-ai/glm5"
  }
}

Oturum içinde model değiştirmek için: /model opus, /model sonnet, /model haiku
