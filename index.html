Tek komutla yapılandırın (önerilen):

npx runcodingplan --add-custom --name "Anthronim" --url "" -a <erişim anahtarınız> -m minimaxai/minimax-m2.5

Başlatmak için:

npx runcodingplan -p anthronim

Diğer modelleri eklemek için:

npx runcodingplan -p anthronim --add-model minimaxai/minimax-m2.7

Detaylar: github.com/ersinkoc/RunCodingPlan

Alternatif olarak ~/.claude/settings.json dosyasını düzenleyin:

{
  "env": {
    "ANTHROPIC_BASE_URL": "",
    "ANTHROPIC_API_KEY": "<erişim anahtarınız>",
    "ANTHROPIC_DEFAULT_OPUS_MODEL": "minimaxai/minimax-m2.7",
    "ANTHROPIC_DEFAULT_SONNET_MODEL": "minimaxai/minimax-m2.5",
    "ANTHROPIC_DEFAULT_HAIKU_MODEL": "z-ai/glm5"
  }
}

Oturum içinde model değiştirmek için: /model opus, /model sonnet, /model haiku
